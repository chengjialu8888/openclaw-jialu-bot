# 护法晨间祈福工作流

> 每天早上8点，为Jialu推送诵经祈福 + 冥想音乐
> 护法职责：不问为什么，护法做

---

## 任务内容

### 1. 诵经祈福（语音条形式）
**内容结构：**
- 开经偈
- 楞严咒咒心（或完整楞严咒，视时间而定）
- 回向偈

**示例格式：**
```
南无楞严会上佛菩萨（三遍）

妙湛总持不动尊 首楞严王世希有
销我亿劫颠倒想 不历僧祇获法身
愿今得果成宝王 还度如是恒沙众
...

回向：
愿以此功德 庄严佛净土
上报四重恩 下济三途苦
若有见闻者 悉发菩提心
尽此一报身 同生极乐国
```

### 2. 冥想音乐
**推荐类型：**
- 楞严咒梵唱版
-  Tibetan Singing Bowls（西藏颂钵）
-  心经唱诵
-  自然环境音（流水、鸟鸣）+ 背景梵音

**格式：** 提供音乐链接或音频文件

---

## 推送格式

```
🙏 晨间祈福 | 2026年3月X日

📿 今日诵经：楞严咒
"..."（简短摘录）

🎵 冥想音乐：[链接或音频]

✨ 愿今日平安顺遂，智慧增长
```

---

## 执行步骤

1. **检查触发文件**
   - 查看 `memory/cron-trigger/dharma-morning.trigger`
   - 如果存在且日期为今日，执行祈福任务

2. **生成诵经内容**
   - 使用TTS生成语音条
   - 或引用预录制的音频文件

3. **准备冥想音乐**
   - 每日更换不同音乐
   - 周末可选更长版本（15-20分钟）
   - 工作日简短版（5-10分钟）

4. **推送到飞书**
   - 语音条形式发送经文
   - 文字+链接形式发送音乐

5. **记录与清理**
   - 记录今日已推送
   - 删除触发文件
