# AI日报生成前自检清单
# 每次生成日报前，必须逐项确认 ✅

## 时间确认
- [ ] 当前北京时间：____年__月__日 __:__
- [ ] 当前美国时间（太平洋）：____年__月__日 __:__
- [ ] 美国东部时间：____年__月__日 __:__

## 搜索参数检查
- [ ] 使用了 `--time-range 1d` 参数
- [ ] 搜索关键词包含当前日期
- [ ] 优先搜索官方来源（公司博客、官方X账号）

## 每篇新闻验证（必须）
- [ ] 原始发布时间是否标注明确？
- [ ] 是否在24小时内？
- [ ] 是否为官方来源或权威媒体？
- [ ] 是否交叉验证了至少2个独立信源？
- [ ] 内容是否为原创而非转载？

## 常见陷阱检查
- [ ] 排除了网页索引/爬虫时间
- [ ] 排除了"回顾"、"盘点"类文章
- [ ] 排除了未标注日期的文章
- [ ] 排除了论坛帖子/自媒体猜测

## 质量控制
- [ ] 今日新闻数量≥3条且已验证
- [ ] 旧新闻比例<10%
- [ ] 每条新闻都有明确来源链接
- [ ] 对存疑新闻标注了"⚠️ 待验证"

## 输出前最终确认
- [ ] 如果新闻不足，已准备诚实说明而非填充旧闻
- [ ] 没有将11月/12月/1月/2月的旧闻当作今日新闻
- [ ] 没有将"即将发布"、"计划"当作"已发布"

---

**如果发现自检不通过，请：**
1. 停止生成日报
2. 重新搜索，扩大关键词范围
3. 或输出："今日重大AI动态有限，主要新闻集中在X月X日"
4. 绝不编造或混入旧闻

**历史教训：**
- 2026-03-03：因未严格核实，将2025年11月的千问、灵光APP误认为3月新闻
- 2026-03-03：将网页索引时间误认为新闻发布时间
